# GITHUB.SIGNAL

An AI-powered GitHub trending dashboard with a bold constructivist / futurist UI.

## Features

- Live GitHub trending discovery
- Bilingual EN/ZH interface
- Domain filtering for focused browsing
- Bilingual repo summaries
- AI commercialization analysis
- One-click jump to GitHub repositories
- Light / dark theme switching

## Why this exists

This project turns GitHub trending data into a visually striking, easy-to-scan daily signal board. It is designed for people who want to quickly discover what is hot, what it does, and whether it might have commercial potential.

## Run locally

```bash
python server.py
```

Open `http://localhost:8090` in your browser.

## AI / privacy note

Sensitive API credentials are not stored in the repository. Set `DEEPSEEK_API_KEY` in your environment before using AI analysis.
